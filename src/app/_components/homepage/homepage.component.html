About us

Nasa API hub

Nasa API Hub is non profit application created for the sole purpose of exploring Nasa's open API systems & different development enviroments. It is invisioned for this application to display data served through couple of Nasa's API systems & present the data about Astronomy picture of the day, Near Earth objects & many more...

Upcoming Features

In the upcoming version of this application, the version 2.0, an additional feature will be added.
A feature which will display Near Earth Objects, where you can see, based on a specific date, whether there were or weren't any hazardous objects in the vicinity of the Earth, alongside with their diameter and date of passing.

In the initial release of the Near Earth objects feature, you will be able to see the aforementioned information based only on a predefined date, when later on, you as a user will have the ability to enter any date you want & the information will be presented to you based on the input you've provided.
