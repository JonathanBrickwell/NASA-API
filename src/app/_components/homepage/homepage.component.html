About

Nasa API hub

Nasa API Hub is non profit application created for the sole purpose of exploring Nasa's open API systems & different development enviroments. It is invisioned for this application to display data served through couple of Nasa's API systems & present the data about Astronomy picture of the day, Near Earth objects & many more...

Mars Rover Feature

Besides the astronomy picture of the day, this app will provide Mars Rover photos!
The idea is to use the Mars Rover API & display a different picture each day from several different cameras mounted on the rover.
Initially we will display only the date & the picture, down the road we will implement the mechanism which will display the information regarding which camera took which photo! Pretty exciting!
